Design a complete premium React Native mobile application for AI-powered menstrual health tracking.

The application includes:

1. Splash Screen (Aurora Gradient)
2. Authentication (Glassmorphism)
3. Home Dashboard (Bento + Glassmorphism)
4. Calendar (Minimal + Material)
5. Four Menstrual Phases (Gradient Cards + Glassmorphism)
6. Cycle Analytics (Dashboard + Bento)
7. AI Chat (Material + Minimal)
8. Educational Videos (YouTube-inspired Cards)
9. Mood Tracking (Soft UI / Neumorphism)
10. AI Prediction (Glassmorphism + Charts)
11. Profile & Settings (Cupertino + Material)

Requirements:
- Consistent premium healthcare branding
- Soft pink, lavender, peach, mint, and white color palette
- Rounded corners (20–28px)
- Glass blur only where appropriate
- Smooth animations and micro-interactions
- Accessible typography (Inter or SF Pro Display)
- Minimalist layouts with generous whitespace
- Elegant charts and progress indicators
- Bottom navigation with Home, Calendar, Analytics, AI Chat, and Profile
- Offline-first design suitable for React Native
- Create a unique 2026 UI that is modern, trustworthy, and calming. Avoid copying existing apps while maintaining a cohesive visual language across all screens.
